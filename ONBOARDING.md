### Skills to Know About
- [/compact](https://docs.claude.codes/ skills/compact) — compresses long conversations to save context. The team's most-used command — sessions run deep and this keeps them going.
- [/model](https://docs.claude.codes/ skills/model) — switch between Claude models mid-session. Useful when you need more power for complex architecture or faster output for simple edits.
- [/code-review](https://docs.claude.codes/ skills/code-review) — review the current diff for bugs, security issues, and cleanups. Used regularly before committing changes to custom nodes.
- [/security-review](https://docs.claude.codes/ skills/security-review) — deep security audit of pending changes. Important for API-key-handling code and HTTP client usage.
- [/review](https://docs.claude.codes/ skills/review) — review a pull request. Used when collaborating on shared ComfyUI node packages.
- [/init](https://docs.claude.codes/ skills/init) — generates a CLAUDE.md file for a new repo. Used once to bootstrap this project's codebase documentation.
